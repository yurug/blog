# yann.regis-gianas.org

This repository hosts the **rendered** static site for [yann.regis-gianas.org](https://yann.regis-gianas.org).

The contents of the `gh-pages` branch are served by GitHub Pages with the custom domain. This repository is updated automatically by CI from the private source repository on each push to `main` of the source.

To read the blog: visit [yann.regis-gianas.org](https://yann.regis-gianas.org).
